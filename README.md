# Student Performance prediction
Developed a predictive model to assess student performance based on academic and socio-economic factors.

Data Preprocessing: Handled missing values, normalized data, and performed feature engineering to improve model accuracy.

Exploratory Data Analysis (EDA): Used Pandas, Matplotlib, and Seaborn to visualize trends and correlations in student performance.

Machine Learning Algorithms: Implemented and compared models such as Linear Regression, Decision Trees, Random Forest, and Support Vector Machine (SVM) for accurate prediction.

Evaluation Metrics: Measured model performance using Mean Squared Error (MSE), R² Score, Accuracy, and Confusion Matrix.

Hyperparameter Tuning: Optimized models using techniques like GridSearchCV and RandomizedSearchCV to improve prediction accuracy.

Technology Stack: Python, Scikit-learn, NumPy, Pandas, Matplotlib, Seaborn.

Results & Insights: Identified key features impacting student performance, providing data-driven insights for academic improvement.
